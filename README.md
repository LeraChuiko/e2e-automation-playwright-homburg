# Playwright Homburg Automation

Automated testing project for the https://termine-reservieren.de/termine/homburg/ appointment booking system.

## 🚀 Tech Stack
- **Playwright** (Test Automation Framework)
- **JavaScript**
- **Git** (Version Control)

## 📁 Project Structure
- `tests/` — Test files (specs) covering the booking flow.
- `helpers.js` — Custom helper functions to encapsulate locators and reusable actions.

## 🛠 Features
- Automated end-to-end booking flow.
- Step-by-step indicator validation.
- Dynamic Übersicht (summary) table verification.
- Error handling for unavailable time slots.

## 🛠 How to run
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run tests: `npx playwright test`
